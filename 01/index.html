학습 시작하기

1. 인공지능 편향의 두 가지 원인

학습 데이터 편향

인간이 만든 불균형한 데이터를 학습할 때 발생합니다. 특정 집단에 대한 정보가 부족하면 AI는 차별적인 판단을 내리게 됩니다.

알고리즘 편향

데이터를 처리하는 규칙(수식)에 설계자의 의도나 사회적 고정관념이 투영될 때 발생하며, 편향을 구조적으로 재생산합니다.

인간 vs 인공지능의 편향 차이

비교 항목 인간의 편향 인공지능의 편향
근본 이유 경험, 문화, 교육 등 환경 학습 데이터와 알고리즘
수정 가능성 교육과 자각을 통해 수정 데이터/알고리즘 기술적 변경
영향력 개인/집단의 선택에 영향 시스템 전반에 광범위한 영향
